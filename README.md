# Penjelasan Program
Ini adalah game flappy bird, dimana player harus menerbangkan burung melewati pipa-pipa. 

Saat pertama kali membuka aplikasi, player akan disambut dengan homescreen dengan tombol untuk memulai permainan atau keluar dari permainan. Ketika tombol mulai permainan dimulai maka permainan dimulai.

player menerbangkan burung dengan cara menekan spasi berulang kali. setiap kali player melewati pipa player akan mendapatkan satu poin.

Apabila burung menabrak pipa atau jatuh ke tanah maka permainan berakhir (game over) dan player akan ditampilkan layar gameover. di layar ini akan ditampilkan skor juga tombol untuk restart (player juga bisa menekan 'R' untuk restart)
